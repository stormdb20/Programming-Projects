CTEC TRACKS

Network Enterprise Infrastructure Track

Gain skills in cloud computing, data management and infrastructure management. Students in this track use hardware and software to build networks that enable connectivity, communication, operations and management.
  • CTEC 335 - Network Protocols (TCP/IP) 3 Credits
  • CTEC 405 - Server Adminstration II 3 Credits
  • CTEC 410 - Internet Technology 3 Credits
  • CTEC 435 - Unix System Administration 3 Credits
  • CTEC 475 - Cloud Computing 3 Credits

Internet Technologies Track

In this track, students take a deep dive into edge technology by learning how to configure sensors, wireless network connections and edge-computing platforms. Courses in this track prepare students to design and program prototypes and systems using the Internet of things (IoT).
  • CTEC 293 - Edge Technology 3 Credits
  • CTEC 415 - Edge Computing 3 Credits
  • CTEC 425 - Edge to Edge Technology 3 Credits
  • CTEC 465 - On the Edge: The Internet of Things 3 Credits

Data Science and Database Administration Track

Students use technical skills to predict and describe prescriptive modeling to support decision-making. In addition, students will implement and design the architectural structure of the Oracle Database Management System.
  • CTEC 298 - Symbolic Computation using Big Data 3 Credits
  • CTEC 300 - Database Architecture 3 Credits
  • CTEC 324 - Database Application Development 3 Credits
  • CTEC 424 - Database Administration 3 Credits
  • CTEC 426 - Expert Systems using Big Data 3 Credits

Health Technology & Cybersecurity Track

The healthcare field has transformed during Covid19 to extend its services to the edge of the network. This track is a new approach which is a combination of healthcare, network, and cybersecurity backgrounds to manage, detect, and secure the privacy of healthcare data through the following: email security, physical security, security controls and risk assessments.
  • BIOL 311 - Hum Anat & Physlgy I 4 Credits
  • BIOL 312 - Hum. Anat. & Physiology II 4 Credits
  • IDIS 240 - Medical Terminology 3 Credits
  • CTEC 477 - Computer and Network Digital Forensics 3 Credits
  • CTEC 300 - Database Architecture 3 Credits
